Fern Key Questionnaire Answer Ask fern keys questions to narrow down fern species match in database and return it.
FUNCTION IdentifyFern() ASK Question1 GET Answer1 IF Answer1 leads to GroupA THEN ASK Question2A GET Answer2A IF Answer2A leads to SubgroupA1 THEN ASK Question3A1 GET Answer3A1 IF Answer3A1 == YES THEN RETURN FernTypeA1 ELSE RETURN FernTypeA2 END IF ELSE RETURN FernGroupA Result END IF ELSE IF Answer1 leads to GroupB THEN ASK Question2B GET Answer2B IF Answer2B leads to SubgroupB1 THEN RETURN FernTypeB1 ELSE RETURN FernGroupB Result END IF ELSE RETURN "Unable to identify fern." END IF END FUNCTION

Fern Image Match fern image against fern training images in database to find a match and return it.
FUNCTION MatchFernImage() BestMatchScore ← 0 BestMatchFern ← None INPUT UserFernImage FOR TrainingImage IN Database COMPARE MatchScore between TrainingImage and UserFernImage IF MatchScore > BestMatchScore THEN BestMatchScore ← MatchScore BestMatchFern ← TrainingImage.FernSpecies END IF END FOR IF BestMatchFern ≠ None THEN RETURN BestMatchFern ELSE RETURN "Unable to identify fern." END IF END FUNCTION
